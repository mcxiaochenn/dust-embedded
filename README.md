# dust-embedded

一个高中生的嵌入式学习记录——从点灯开始，一步步走向硅与代码的深处 🚀

## 🛠️ 开发环境

- **ESP-IDF** — ESP32 系列开发
- **STM32CubeIDE** — STM32 开发
- **PlatformIO** — 跨平台嵌入式开发

## 📁 项目结构

```
dust-embedded/
├── esp32/          # ESP32 相关项目
├── stm32/          # STM32 相关项目
├── docs/           # 学习笔记与文档
└── tools/          # 常用工具与脚本
```
